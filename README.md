 # 🌤️ Weather App - Open-Meteo API

A beautiful, responsive weather application that provides real-time weather data and forecasts using the free Open-Meteo API. No API key required!

## ✨ Features

- 🔍 **City Search** - Search any city worldwide
- 🌡️ **Current Weather** - Temperature, feels like, humidity, wind speed, precipitation
- 📅 **3-Day Forecast** - Daily high and low temperatures
- 🎨 **Weather Icons** - Visual representation of weather conditions
- 📍 **Geolocation** - Get weather for your current location
- ⚡ **Fast & Responsive** - Works on all devices (mobile, tablet, desktop)
- 🔄 **Loading States** - Smooth loading animations
- ⚠️ **Error Handling** - User-friendly error messages
- 💾 **Caching** - Reduces API calls with local caching
- 📡 **Offline Support** - Shows cached data when offline
